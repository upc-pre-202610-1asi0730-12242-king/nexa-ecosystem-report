## 5.2. Landing Page, Services & Applications Implementation

Esta sección documenta el avance incremental de la Landing Page, la Web Application, el soporte simulado de servicios, la base backend, la evidencia de despliegue y la colaboración del equipo. Sprint 1 corresponde a la línea base AV1 y concentra la Landing Page pública, la estructura Docs-as-Code y la evidencia inicial de discovery/diseño. Sprint 2 corresponde a TB1 y concentra el incremento de Web Application, los flujos S1/S2, la planificación S3 y el soporte simulado para validar recorridos frontend. Sprint 3 corresponde al corte AV2 frontend/backend, incorporando `nexa-website v3.0.0`, `nexa-webapp v2.0.0`, `nexa-platform v1.0.0`, despliegue controlado en Render, Swagger/OpenAPI, User Flow S3, Wireflow S3 y configuración/migración de backend hacia PostgreSQL.
