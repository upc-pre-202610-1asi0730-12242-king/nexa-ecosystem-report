# Registro de Versiones del Informe

El presente registro documenta las modificaciones significativas del informe académico Nexa para AV1, TB1 y AV2. Para mantener trazabilidad académica, las versiones se agrupan por hitos de entrega y se expresan con formato SemVer asociado al corte documental del reporte. Las referencias a repositorios técnicos se incluyen únicamente cuando la auditoría final permite sustentar versiones, commits o despliegues reales.
