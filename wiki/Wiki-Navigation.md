# Wiki Navigation

This page describes how to navigate and maintain the Nexa wiki pages.

## Layout Overview
- **Sidebar**: Located at `wiki/_Sidebar.md`. Tracks categories (Start Here, Engineering, Development, API, Quality). Update it whenever you create a new guide page.
- **Footer**: Located at `wiki/_Footer.md`. Shows standard copyright and course information.
- **Home**: Located at `wiki/Home.md`. The main entry point.

## Customization Rules
- When writing a new guide page, use standard kebab-case naming for the file (e.g. `My-New-Guide.md`).
