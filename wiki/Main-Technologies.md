# Main Technologies

The Nexa ecosystem integrates several modern tech stacks to deliver premium performance:

## Frontend (WebApp & Website)
- **HTML5 & CSS3**: Structured markup and vanilla styling.
- **Vue 3 (Composition API)**: Central client framework for modular, reactive UI.
- **PrimeVue 4**: Rich component library for dashboards.
- **Pinia 2**: In-memory caching layer acting as an Anti-Corruption Layer (ACL).
- **Vite 5**: Fast build system and bundler.

## Backend (Platform API)
- **C# / .NET 8**: Structured backend programming.
- **ASP.NET Core Web API**: RESTful API service endpoints layer.
- **Entity Framework Core 9**: Object-Relational Mapper (ORM) for data management.
- **PostgreSQL**: Relational database persistence.
