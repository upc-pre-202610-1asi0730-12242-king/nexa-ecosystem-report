# Release Notes

This folder documents the SemVer tags used by the Nexa ecosystem and academic repositories.

## Policy

- Tags stay pure SemVer: `vMAJOR.MINOR.PATCH`.
- Releases describe technical milestones, frontend/backend integrations, and report updates.
- Source archives are stored under the release assets.
